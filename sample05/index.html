Staggerに詳細な指定を行う function

Only use this if you need to run custom logic for distributing the staggers.
これは、スタッガーを分散させるためのカスタムロジックを実行する必要がある場合にのみ使用します。
スタッガーへの関数指定は、 各トゥイーンへ分散した カスタムロジックを実行する必要がある 場合にのみ使用します。

The function gets called once for each target/element in the Array and should return the total delay from the starting position (not the amount of delay from the previous tween's start time).
この関数は、配列の各ターゲット/要素に対して1回ずつ呼び出され、 開始位置からの遅延の合計を返します（前のトゥイーンの開始時間からの遅延量ではありません）。
関数指定は、配列内の各ターゲット/要素毎に1回ずつ呼び出され、 開始位置からの遅延時間の合計を返します。 ( 前回のトゥイーンが開始された時点からの遅延の合計ではないことに注意 )

The function receives the following parameters: index [Integer] - The index value from the list. target [Object] - The target in the list at that index value. list [Array | NodeList] - The targets array (or NodeList).
この関数は、以下のパラメータ(引数)を受け取ります。
index [Integer] - リストからのインデックス値です。
target [オブジェクト] - リスト内のインデックス値にあるターゲット。
list [Array | NodeList] - ターゲット配列（またはNodeList）。 

      gsap.to(
        targetArray,
        {
          ...,
          /*
          @index ... index of target element which in list
          @target ... target element in list
          @list ... element list which is set stagger
          */
          stagger: function ( index, target, list ){
            const val = index * 10;
            const styleVal = "-" + val + "px";
            target.innerText = styleVal;
            target.style.marginTop = styleVal;
      
          }
        }
      );
