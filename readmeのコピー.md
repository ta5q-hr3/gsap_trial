# GSAPの練習場です。

GSAPのいろいろな試作を行います。

Viteを使ってます。

viteの初期化は
npm create vite@latest
で行うんだったかな？
これだとプロジェクトフォルダを生成するんですよねー....

npm install vite を行って、必要なファイルを揃えた方がよかったかも。
その代わりpackage.json の script は自分で準備する必要あります。
↓↓↓
"dev": "vite",
"build": "vite build",
"preview":  "vite preview",


vite.config.js は自前で準備する必要があるようです。
ポート番号は8888を指定してあります。
